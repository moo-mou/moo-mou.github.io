Is this usable today?

This is very much a work in progress. There is quite a bit of setup involved to get it working right now (install fluidsynth, compiling a custom version of vim to flush buffer, etc). On top of that, the sound generation for vim keystrokes still need a lot of work to make them sound "nice". If you are interested to help out, please do get in touch!
